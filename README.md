# Antigravity Cockpit (Private)

Personal, local-only quota dashboard for Antigravity.

## Scope
- Local client process only (no OAuth, no remote API)
- Single-user workflow (no accounts, no sync)
- No announcements or telemetry

## Features
- Webview dashboard + QuickPick view
- Quota grouping, pinning, renaming
- Status bar monitoring + threshold notifications
- Quota history

## Languages
- English
- Korean
